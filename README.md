# Process
-Analyzed sales data to track key KPIs such as Total Revenue, Orders, Profit, and Sales Growth.
-Cleaned and transformed raw data by handling duplicates, missing values, and creating calculated fields.
-Built an interactive sales dashboard using Pivot Tables, charts, and KPI cards.
-Enabled dynamic analysis by product, category, region, and time using slicers and filters.
-Provided actionable insights to identify top-performing products, regions, and sales trends.
